iTerm2

安装

  1. 下载地址: https://www.iterm2.com/

  2. 将iTerm2设为默认终端

  3. 给iTerm2设置全局快捷键

  4. 配色
  5. 设置窗口
    定位到 [Preferences - Profiles - Window] 新窗口设定风格设为 [Top of Screen]，这样每次新打开终端都会固定出现在屏幕的顶端且不会有窗口边框。如果你也使用多 Space 和双显示器的话，同时选择 Main Screen 和 All Space 会比较方便
  6. 隐藏 Dock 图标
    让我们的终端变得更 Cool，让它来无影去无踪。这一步我要 iTerm 启动后不再出现在 Dock 上，打开终端输入下面的命令，然后重启 iTerm。
配置插件
安装oh-my-zsh

sh -c "$(curl -fsSL https://raw.github.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"

修改主题

安装oh-my-zsh成功后，用vim打开隐藏文件 .zshrc ，修改主题为 agnoster

应用这个主题需要特殊的字体支持，否则会出现乱码情况，这时我们来配置字体:

  1. 使用 Meslo 字体，点开连接点击 view raw 下载字体。
  2. 安装字体到系统字体册
  3. 修改iTerm2的字体

在修改了iTerm后,Mac自带的Terminal显示会出现异常

我们需要进入Terminal的偏好设置中,取消掉 显示ANSI颜色 和 修改字体 保持和 iTerm 的字体一致。

修改后效果,勉强能够接受(不再显示乱码了)
