使用Flutter开发,难免会遇到需要和原生开发进行交互的地方,这个时候我们就需要掌握混合开发的相关技能。

创建Flutter module

在做混合开发之前,我们首先需要创建一个 Flutter module.

假如你的Native项目路径是： xxx/flutter_hybrid/Native项目

上面的终端指令会切换到你的原生项目的上一级目录,并创建一个flutter模块:

  • .android flutter_module的Android宿主工程
  • .ios: flutter_module的iOS宿主工程
  • lib: flutter_module的Dart部分代码
  • pubspec.yaml: flutter_module的项目依赖配置文件

因为宿主工程的存在,我们这个flutter_module在不加额外的依赖配置的情况下也是可以独立的运行的.
