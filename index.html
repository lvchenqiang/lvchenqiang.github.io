谁是谷歌想要的人才？

谁是谷歌想要的人才呢？ 这本书提出了很多有意思的问题，可以让你发散性思维同时内心又意识到收敛性思维的存在。追求问题的真相，很多问题其实又是没有准确答案的,只有更好更值得期待的答案。

Tips

  1. 有创造力的成功人士,大多都是极其聪明的人;而那些极其聪明的人,在事业或生活上既成功又有创造力的却少之又少。
  2. 发散和收敛性思维是阴阳的两面,成功的创新家需要两者兼备。只善于进行发散性思维的人，或许会显得怪头怪脑。而只具备收敛性思维天赋的人，聪明却缺乏创意。
  3. 把你的逻辑在白板上画出来,在期待灵感降临期间，写写画画让你有了些事情可做。白板还能帮助记忆，把你的结论和中间值保留下来。
  4. 通常，一个算法问题不止一种解法。你的第一个可行的设想不一定是最佳的解法。所以要乐于探索替代解法。
  5. 面试时，切记冷场。沉默会让双方感到不舒服。如果你真的不知道该怎么解决问题,可以采取下面的一些建议: 复述问题 消除歧义 说明为什么最一目了然的答案行不通 类比和比喻 头脑风暴 评判

有趣的题目

如何让世界最挑剔的公司选中你

  1. 你被缩小到一枚硬币大小，扔进搅拌机。你的质量减少，密度保持不变。搅拌机60秒内就会开始搅拌，你会怎么做？
    答案:跳出搅拌机
    推理过程:我们依稀应该还记得 火箭的能量公式: E = mgh.E 是能量，m是质量， g是重力加速度， h是飞起的高度。在质量较少和密度不变的情况下，肌肉能量和质量等比缩小，所以跳跃的高度就会保持不变

  2. 刮风的时候,往返班机会花更长时间、更短时间、还是相同时间？
    答案:会用更长的时间

  3. 你把一杯水放在电唱机的转盘上，并开始慢慢的提速。最先发生哪件事:杯子划出转盘，翻倒？ 还是水溅出来？
    答案:取决于杯子的形状，以及它装的水有多满。

    答案:要来回11趟完成,具体的分析如下:
    推理过程:人可以不下船，将狮子退下船，也可以不下船将狮子诱捕上船。

  2. 用一个能计时4分钟的沙漏和7分钟的沙漏，精确计时9分钟。
    答案:两个沙漏同时打开,当七分钟时，翻转7分钟的沙漏。这时，4分钟的沙漏已经用完一次并翻转了，而且还有一分钟的沙。而这一分钟的沙漏，可以让7分钟的沙漏漏一分钟的沙，于是我们就有了9分钟。

  3. 在一个黑暗的房价里面，有人塞给你一叠牌，有N张面朝上，其余的面朝下。你看不见牌，怎么才能把牌分成两堆，让每一堆面朝上的数目相同呢？
    答案:先数出N张牌，并将其翻转,剩下的是另一堆。这两堆牌，向上的数目是一样的。

  4. 你坐在一辆车上，车里面栓着一个氢气球，车窗都关着。当你踩油门前进时，气球会出现哪种情况:它会朝前动，朝后动？ 还是静止不动？
    答案:气球会向前移动
    推理过程:我们普遍的直觉可能是物理都具有惯性，但是我们需要注意的是气球的质量要小于空气。

为什么刁钻的面试题成了主流

  1. 据一项调查显示,70%的人喜欢喝咖啡，80%的人喜欢喝茶。同时喜欢喝咖啡和茶的人，最多有多少？最少有多少？
    答案: 最多70%，最少50%
    推理过程:图形分析。

  2. 在3:15,表盘的分针和时针会是多少度的夹角？
    答案:7.5度
    推理过程:360 1/12 1/4 = 7.5

  3. 在1和1000之间有多少个包含了“3”的整数?
    答案:271个
    推理过程:C31 A91A91 + C32 * A91 + C33 = 243+27+1 = 271

  4. 一本书有N页,按常见的方式，从1到N编了号。页码上单个数字的总数是1095.这本书有多少页?
    答案:401页
    推理过程: N+(N-9)+(N-99) = 1095

  5. 100的阶乘最后会有多少个0？
    答案:24个0
    推理过程:我们知道每个0,其实就是(2*5),所以我们看100以内有多少个5的倍数。

揭秘谷歌的招聘机制

  1. 在一条人烟稀少的高速公路上，30分钟里看到一辆车的概率是95%，那么，10分钟内看到一辆车的概率是多少？
    答案:约63%
    推理过程:我们计算的真正的意义是，30分钟内没有车通过的概率。假设10分钟内有车通过的概率是x,那么
    (1-x)3 = 0.05;

  2. 以下两种条件，你可以任选其一。1 给你一个篮球，你只有一次机会投篮，投中则得奖。 2 让你投篮三次，有两次投中则可以得奖。
    答案:糟糕的选手选1，优秀的选手选2.

  3. 楼梯有9阶台阶，一步可以上一个台阶，也可以上两节台阶。那么共有多少种方式到达第9阶台阶？
    答案:55种
    推理过程:斐波那契数列

4.你见过的最漂亮的方程是哪一个？请做说明。
答案:参考费曼的简化方程式： U = 0.
推理过程:
质能方程: E = mc2
广义相对论: G = 8πT G:爱因斯坦张量,代表时空的曲率,T代表应力能量张量,衡量的是质量和能量的密度。方程式是说:质能会扭曲空间和时间，让我们误以为自己体验的是重力。
薛定谔方程: HΨ = EΨ 波函数的哈密尔顿量等于它的能量
欧拉方程:eπi + 1 = 0

为什么传统的工程师思维已经不再流行了

  1. 你如果要确定鲍勃拿到了你的电话号码，但是不能直接问她。你必须在卡片上留言，拿给中间人，伊娃。伊娃会把你的卡片交给鲍勃，鲍勃又把自己的信息交给伊娃，再由伊娃交给你。你又不愿意伊娃知道你的手机号码，那么你会怎么问鲍勃？
    答案: 给出一个RSA答案，并仔细解释着整个过程。
    推理过程:
    N = p*q; 公共秘钥的一部分

    xe mod N = Y; x是电话号码
    // 根据发过来的值计算
    d = e mod (p-1)(q-1);
    Yd mod N = x;
    x即为电话号码。

  2. 如果你有一叠硬币,摞起来有帝国大厦那么高,你能把它们都放进一个房间吗？
    答案:只要这个房间能平铺下100枚硬币,就放得下所有硬币。
    推理过程:帝国大厦差不多是100层高,至少比普通房间的室内高100倍。所以,我们可以假设将100层高的硬币放到一个房间。所以一个房间如果能够平铺100枚硬币，就放得下所有的硬币。

洞悉面试官遴选人才的思路

  1. 一名男子把车推到酒店,损失了财富。这是怎么回事？
    答案:他在玩大富翁

  2. 你在山脚下坐上滑雪缆车,一路攀升到山顶。你经过的缆车车厢，占总车厢数的几分之几？
    答案:你会经过所有缆车车厢,当然 除了你自己坐的那一个。
    推理过程:由于车厢的速度是滑轮的两倍,你其实通过了整个缆线圈。因此,也就经过了除了你自己之外的所有缆车车厢。

  3. 请看如下的序列,请问下一行应该怎样的？

         1
    1  1
    2  1
  1 2  1 1
1 1 1 2 2 1

    答案: 3 1 2 2 1 1
    推理过程:“看读序列”,下一行都是对上一行的解读。

  4. 你有25匹马,需要多少场比赛才能找出跑的最快的3匹马？你没有秒表，每场比赛只能同时让5匹马参赛。
    答案: 7 场

如何回答 “你会怎样…..” 类问题

  1. 下雨了，你必须赶到停车场最远处的汽车那儿。如果以尽量不被淋湿为目标，是跑过去还是走过去？
    答案:有伞的时候,走过去。 没伞的时候,跑过去。

  2. 你有一只卡车队,有50辆车，每辆车配备了满缸汽油,能行驶100英里。你总计能行驶多远？ 如果你有N辆车呢？
    答案: 有50辆车,可行驶449.92英里。如果N辆车,有效的行程为 100*(1+1/2+1/3+1/4+……+1/N).

  3. 你有无限供给的砖块,要把砖块堆起来,每块砖都要从其下的一块砖突出一部分。以这样的方式堆叠，最多能悬空多长距离？
    答案: 大约能悬空两块砖
