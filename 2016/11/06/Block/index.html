Block 详解

发表于 2016-11-06 | 分类于 iOS ， BOOK |

Block 概要

Block 是C语言的扩充功能。用一句话表示这个扩充功能，就是带有自动变量的匿名函数。

匿名函数，就是所谓的不带有名称的函数。C语言的标准中不允许存在这样的函数，必须使用函数的名称或者函数指针来进行调用。而通过Block，源代码中就能够使用匿名函数。

”带有自动变量的匿名函数“这一概念并不仅指Block,它还用于其他许多程序语言中。
例如:

程序语言 Block名称
C+Blocks Block
Smalltalk Block
Ruby Block
LISP Lambda
C+Python Lambda
C++11 Lambda
JavaScript Anonymous function

Block 模式

Block语法

Block表达式语法与一般的C语言函数定义相比，仅有两点不同。

  1. 没有函数名
  2. 带有”^”

第一点没有函数名，因为是匿名函数。第二点不同是返回值类型前带有”^”(插入记号)记号，便于在大量是的使用中便于查找。

^ 返回值类型 参数列表 表达式

当然在使用中也可省略为:

^ 参数列表 表达式 /// 无返回值类型

^(int count){
return (count+1);
}

^ 表达式 ///省略返回值和参数列表

^{

printf(“Block\n”)

}

Block 类型变量

与使用函数指针的源代码对比可知，声明Block类型变量仅仅是将声明函数指针类型变量的”*“变为”^“。Block的类型变量则与C语言类型变量相同，可作为以下用途:

  • 自动变量
  • 函数参数
  • 静态变量
  • 静态全局变量
  • 全局变量

截获自动变量值

Block语法的表达式使用的是它之前声明的自动变量fmt和val. Block中，Block表达式截获所使用的自动变量的值，即保存该自动变量的瞬间值。这既是截获自动变量值。

__Block说明符

自动变量值截获只能保存执行Block语法瞬间的值，保存后就不能改写该值，如果想要改写截获的自动变量，则需要使用到__block.

使用附有block 说明符的自动变量可在Block中幅值，该变量称为block变量。
