Block 概要

Block 是C语言的扩充功能。用一句话表示这个扩充功能，就是带有自动变量的匿名函数。

匿名函数，就是所谓的不带有名称的函数。C语言的标准中不允许存在这样的函数，必须使用函数的名称或者函数指针来进行调用。而通过Block，源代码中就能够使用匿名函数。

”带有自动变量的匿名函数“这一概念并不仅指Block,它还用于其他许多程序语言中。
例如:

程序语言 Block名称
C+Blocks Block
Smalltalk Block
Ruby Block
LISP Lambda
C+Python Lambda
C++11 Lambda
JavaScript Anonymous function
